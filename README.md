# IASO_DL_individual
Repository with my Modal app for my plant pathology individual project. The result plots and csv files are not included. 

## __init__ 
this file makes the project work like a package, as mentioned in https://modal.com/docs/guide/project-structure 
I followed those steps to make my app work 

## app.py 
this file defines the app and runs the main function. Note that I only ran once the preprocessing and I used the training and the evaluation multiple times for adjustments in the code. 

## evaluate_app, preprocessing_app, train_app
these are the files with the functions I ran in app.py 

## models.py 
this file has the model architectures as classes and the necessary functions to run them 

## requirements.txt 
I used my deep learning environment with all the add ons for this project 

## DL_plots 
it has a couple plots that I used to visualize the results
